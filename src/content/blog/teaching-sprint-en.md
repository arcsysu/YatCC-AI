To keep the homepage and teaching practice section aligned, course updates, lab notices, and platform changes can now be published from the same blog directory.

That gives us a few immediate benefits:

- one consistent authoring workflow
- no manual homepage timeline editing
- richer detail for each announcement

Adding a new update is now as simple as creating one markdown file with frontmatter.
